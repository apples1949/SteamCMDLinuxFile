# SteamCMDLinuxFile
通过白嫖github下载更新好的steamcmd程序。需要的人自行复刻仓库然后手动运行流程。最后在build文件夹中下载。最好是获取第三方加速链接 https://github.com/xiaoxuan6/github-mirror  
常用指令:  
```
mkdir steamcmd
```
```
cd ./steamcmd
```
```
wget https://raw.kkgithub.com/apples1949/SteamCmdLinuxFile/main/build/steamcmd_linux.tar.gz.zip
```
```
tar -xzf steamcmd_linux.tar.gz
```
```
bash steamcmd.sh
```
