# SteamCMDLinuxFile
通过白嫖github下载更新好的steamcmd程序。需要的人自行复刻仓库然后手动运行流程。最后在build文件夹中下载。记得获取第三方加速链接 https://github.com/xiaoxuan6/github-mirror  
常用指令:  
```
mkdir steamcmd
```
```
cd ./steamcmd
```
```
wget https://raw.kkgithub.com/apples1949/SteamCmdLinuxFile/main/build/.zip
```
别直接复制↑仅参考  
```
tar -xzf 
```
